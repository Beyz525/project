What is

Marxstreet has come up with a rug check feature solution, combining the Defi platform to minimize fraud against crypto users, especially those who are still beginners.

Frequently Asked Questions

  • Top Questions

What is Marxstreet ?

marxstreet is a defi protocol that runs on the solana network.

What are the features of Marxstreet?

Marxstreet features generally have the same features as defi in general, but what differentiates marxstreet from other defi is that marxstreet is equipped with a rug check system.

What is the function of rug check on Marxstreet?

The rug check function on Marxstreet is to find out the functions contained in the smart contract of a token, in order to minimize fraud against crypto users.

What type of security does MarxStreet use in its Rug Check system?

Marxstreet uses AI with the latest system to read and analyze all the contents of the smart contract through a scanning system.

What problem is Marxstreet trying to solve?

As the blockchain era develops, there are quite a few developers who commit fraud against crypto users, especially users who are new to the world of crypto. Not everyone can read the contents of a smart contract, so MarxStreet was born with a rug check system to make it easier for users to read and understand the function of the smart contract.

How do I use the rug check feature?

To use the rug check feature, users can enter the Marxstreet dapps menu, then click the rug check menu, then enter the smart contract they want to analyze.

on which networks can users use rug check?

For now, Rug Check can still be used on the Solana network.

Whom can I reach out to with other questions?

Users can contact Marxstreet via all social media or via the email that has been prepared.
